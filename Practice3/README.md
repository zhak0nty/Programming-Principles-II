# Practice 03

## How to run
From the repository root:

### Run a specific file
python3 "Practice 03/functions/basic_functions.py"
python3 "Practice 03/lambda/lambda_basics.py"
python3 "Practice 03/classes/class_definition.py"
python3 "Practice 03/inheritance/inheritance_basics.py"

## What is inside
- functions/   -> examples of functions, arguments, return values, *args/**kwargs
- lambda/      -> lambda basics + map/filter/sorted
- classes/     -> class basics, __init__, instance methods, class methods, variables
- inheritance/ -> inheritance basics, super(), overriding, multiple inheritance
